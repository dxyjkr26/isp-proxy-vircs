## ISP 代理是什么？和普通住宅代理有什么区别？

**ISP 代理**，全称 Internet Service Provider 代理，也叫静态住宅代理，是指 IP 地址由真实网络运营商（如 AT&T、Comcast、Frontier）直接分配给住宅用户的代理服务。这类 IP 在各大平台的风控系统中，识别特征与普通居民上网行为完全一致，被封禁或触发验证的概率远低于数据中心 IP。

普通"住宅代理"有时指的是共享流量代理池——你用的 IP 同一时间可能有几十个人在用，历史污染严重。ISP 代理通常是独享静态 IP，同一个 IP 只给你用，归属干净。

简单说：ISP 代理 = 运营商直接分配 + 住宅归属 + 静态独享。
